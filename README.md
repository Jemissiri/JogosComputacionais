# [Chasing Cars]

A Unity game project.

## Download & Import Instructions

### Download the Project

1. On this GitHub repository page, click the green **"Code"** button
2. Select **"Download ZIP"** from the dropdown menu
3. Extract the ZIP file to a folder on your computer (right-click > Extract All)

### Import to Unity

1. **Open Unity Hub**
   - Launch Unity Hub from your computer

2. **Add the Project**
   - Click the **"Add"** button in the left right conner
   - Click the **"Add project from disk"**
   - Navigate to the extracted folder
   - Select the folder and click **"Select Folder"**/**"Open"**

3. **Open the game**
   - Click the imported project

3. **Open the Main Scene** (if not already open)
   - Go to **Assests > Scenes**
   - Select the main scene > `UpTheSlope`
